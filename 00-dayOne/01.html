First Day Notes

NOTES

  • When using Emmet, class is denoted by a "." and id is denoted by "#"
  • HTML stands for hypertext markup language
  • DOCTYPE states the type of document, in this case HTML
  • our html tag denotes our language to be english, per the lang="en"
  • The head tag houses metadata, or detailed data about our doc.
  • UTF-8 specifice character encoding. Each character (letter, number, even spaces) has a numberic value.
  • viewport notes the device we are reading our document on.
  • title reflects the name that displays on our browser tab.
  • link allows us to connect to other files, such as our stylesheet (CSS)
  • A style tag can help us inject css into our document, but this is not best practice.
  • The body tag houses all the other tags that we want to display onto our browser. This will be what the user sees!
